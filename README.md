## About 👩‍💻 <a id="about"></a>
This project was built as an extension of an in-class Counter component. The goal was to reinforce state-driven UI, DOM manipulation inside a class, and separation of concerns through a documented demo page.

## Features 📋 <a id="features"></a>
⚡️ Fully Responsive<br/>
⚡️ Valid HTML5 & SASS/CSS3<br/>
⚡️ GSAP Scroll Animations<br/>
⚡️ ES6 Modules<br/>
⚡️ OOP Inheritance with StepCounter<br/>

## Technologies ⚙️ <a id="technologies"></a>
JavaScript, HTML5, SASS/CSS3, GSAP

- Open `index.html` in your browser or run with a local server
