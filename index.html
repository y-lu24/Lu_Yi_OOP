Overview

What the Counter class does

The Counter class is a self-contained UI component written in JavaScript. It creates and manages its own HTML elements — a display, an increment button, a decrement button, and a reset button — and mounts them into any container element you provide. Each counter tracks its own count value independently, and updates the display automatically whenever the state changes.
