Overview

What the Counter class does

The Counter class is a self-contained UI component written in JavaScript. It creates and manages its own HTML elements — a display, an increment button, a decrement button, and a reset button — and mounts them into any container element you provide. Each counter tracks its count independently and automatically updates its display whenever its state changes.

What problem it solves

Without OOP(Object-oriented programming), building multiple counters on the same page means repeating DOM manipulation and event listener code for every single instance. The Counter class encapsulates all of that logic in one place — you instantiate as many counters as you need with a single line each, and they all work independently without interfering with each other.

How To Use It

HTML

<div id="counter-container1" class="counter"></div>

JavaScript

import { Counter, StepCounter } from './modules/counter.js';

const counter1 = new Counter("#counter-container1");
const counter2 = new Counter("#counter-container2", 5);
const counter3 = new Counter("#counter-container3", 10);
const counter4 = new StepCounter("#counter-container4", 0, 5);

Parameters

Counter (selector, initialValue)

selector — required. A CSS selector string matching the container element, e.g. "#counter-container1".

initialValue — optional. The starting count value. Defaults to 0.

StepCounter (selector, initialValue, step)

Extends Counter. Accepts all the same parameters plus one more.

step — optional. The amount to increment or decrement by. Defaults to 1.
